**Tomcat** is a server used to run Java web applications like Servlets and JSP.
### Steps
* Install Tomcat
* Start it using scripts in the `bin` folder
* Open in browser using `http://localhost:8080`
* Deploy apps by placing them in the `webapps` folder
### Configuration
* `server.xml` → port settings
* `web.xml` → servlet configuration
### Key Idea
Install → start → deploy → configure.
