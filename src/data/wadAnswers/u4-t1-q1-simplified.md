**Java Servlets** are programs that run on a server to handle user requests and generate dynamic web pages.
### Working
* The **client (browser)** sends a request
* The **web server** forwards it to the **servlet container**
* The container loads the servlet if needed
* Methods like `doGet()` and `doPost()` process the request
* The servlet sends a response back to the client
### Flow
client → web server → servlet container → servlet → response → client
### Key Idea
Servlets act as a **bridge between the client and server-side logic**, enabling dynamic web application development.
