Servlets use a few main methods to handle web requests.
### Methods
* `init()` → runs once for setup
* `service()` → handles every request and decides which method to call
* `doGet()` → used to retrieve data
* `doPost()` → used to send data
* `destroy()` → runs once to clean up resources
### Key Idea
These methods together manage **initialization, request handling, and cleanup** in a servlet.
