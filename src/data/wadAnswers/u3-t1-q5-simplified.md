**XPath**, **XQuery**, and **FLOWR** are used to get data from XML documents.
### XPath
XPath is used to find elements using path expressions like:
```xml
/student/name
```
This selects the **name** element inside **student**.
### XQuery
XQuery is a more powerful language that uses XPath to perform advanced operations like **filtering**, **sorting**, and **retrieving specific data**.
### FLOWR
FLOWR is a structure used in XQuery to process data step by step:
* **for** → loops through elements
* **where** → filters data
* **order by** → sorts data
* **return** → shows the result
### Key Idea
**XPath selects data**, **XQuery processes it**, and **FLOWR organizes how the query is written**.
