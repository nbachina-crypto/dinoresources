**JSP** is used to create dynamic web pages by combining **HTML and Java code**.
### Features
* Built-in objects like `request`, `session`
* Expression Language for easy data access
* Tag libraries to reduce coding effort
### Working
JSP pages are automatically converted into **servlets** by the server, so developers do not need to manage low-level processing.
### Advantages
* Faster development
* Cleaner and more readable code
* Easier maintenance
### Key Idea
JSP makes it easier to build **dynamic web pages** by separating design and logic, reducing the need for complex servlet code.
