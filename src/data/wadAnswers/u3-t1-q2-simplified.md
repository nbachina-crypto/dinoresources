**DTD** and **XML Schema** are used to define rules for an XML document so that data is structured correctly.
### DTD(Document Type Defintion)
DTD specifies:
* What elements and attributes are allowed
* How elements are arranged
  It is simple but has **limited features** and does not support data types properly.
### XML Schema
XML Schema is more advanced. It:
* Defines structure like DTD
* Supports **data types** (string, number, date)
* Allows rules like **value limits**
### Key Idea
Both are used for **validation**, but **XML Schema is more powerful and flexible than DTD**.
