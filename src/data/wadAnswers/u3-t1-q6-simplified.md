**JSON** is a lightweight format used to store and transfer data using **key-value pairs** and **arrays**.
Example:
```json
{
  "name": "Rahul",
  "rollno": 101
}
```
### Features
* Easy to **read and write**
* **Compact** and small in size
* Supported by many programming languages
* Commonly used in **web APIs**
### JSON vs XML
* **JSON** is simpler and shorter
* **XML** uses tags, so it is more detailed but longer
* JSON is faster for **data transfer**, while XML is better for **structured and validated documents**
### Key Idea
**JSON is lightweight and efficient**, while **XML is more powerful for structured and validated data**.
