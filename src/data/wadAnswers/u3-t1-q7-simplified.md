JSON stores data using **simple data types, objects, and arrays**.
### Data Types
* String → text
* Number → numeric value
* Boolean → true/false
* null → empty value
### Objects
Objects store data in **key-value pairs**:
```json
{
  "name": "Rahul",
  "rollno": 101
}
```
### Arrays
Arrays store **multiple values** in a list:
```json
{
  "skills": ["HTML", "CSS", "JavaScript"]
}
```
### Key Idea
**Objects store structured data**, **arrays store lists**, and **data types define the values** used in JSON.
