**AngularJS** is used to build dynamic web applications in the browser.
### Expressions
Expressions display data in HTML using `{{ }}`:
```html
{{ user.name }}
```
They automatically show values from the application.
### Modules
Modules organize the application and act as a container for components:
```javascript
var app = angular.module("myApp", []);
```
### Key Idea
**Expressions connect data to the view**, and **modules organize the application structure**.
