Web development frameworks are mainly of three types.
### Types
* **Client-side frameworks** → run in the browser and handle UI (React, Angular)
* **Server-side frameworks** → run on the server and manage logic and data (Spring, Django)
* **Full-stack frameworks** → handle both frontend and backend (Next.js, Rails)
### Other Types
* **MVC frameworks** → separate data, logic, and interface
* **Microframeworks** → lightweight with fewer features
### Key Idea
Frameworks differ based on **where they run and how they organize application development**.
