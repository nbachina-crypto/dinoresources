
**JDBC** is used to connect Java web applications to a database.
### Steps
* Load the driver
* Connect to the database
* Execute SQL queries
* Process results
* Close the connection
### Usage
* In **Servlets**, JDBC code is written inside `doGet()` or `doPost()`
* In **JSP**, it can be written directly but is usually handled by Servlets
### Key Idea
**Load → connect → query → process → close**, enabling database operations in web applications.
