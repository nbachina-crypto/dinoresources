### What is ReactJS?
**ReactJS** is a JavaScript library used to build **dynamic user interfaces**.
---
### Components
React divides the UI into small reusable parts called **components**.
```javascript
function Welcome(props) {
    return <h1>Hello, {props.name}</h1>;
}
```
---
### Virtual DOM
React uses a **Virtual DOM** to update only the parts of the page that change, making it faster.
---
### Final Idea
* Build UI using **reusable components**
* Use **Virtual DOM** for fast updates
* Manage and display data efficiently using React
