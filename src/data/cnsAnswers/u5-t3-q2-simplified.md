Firewall configuration means setting **rules to control network traffic**.
---
## Main Components
* **Rules** → allow/block traffic
* **Default policy** → usually deny all
* **Zones** → internal, external, DMZ
* **NAT** → hides internal IPs
* **Stateful tracking** → allows valid responses
* **Logging** → records activity
---
## Example Rules
* Allow web traffic (ports 80, 443)
* Block Telnet (port 23)
* Allow internal users to access internet
* Block direct external access to internal network
---
> **Key Idea:** Proper firewall configuration enforces strict control over network traffic, ensuring security while allowing necessary communication.
