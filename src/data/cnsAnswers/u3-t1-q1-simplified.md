The Division Algorithm states:
```text
a = bq + r
```
where:
```text
0 ≤ r < b
```
* Every integer can be expressed in this form
* The remainder is always **non-negative** and **less than the divisor**
---
## Quick Examples
### Example 1
```text
17 = 5 × 3 + 2
```
* Quotient = **3**
* Remainder = **2**
---
### Example 2
```text
20 = 4 × 5 + 0
```
* Quotient = **5**
* Remainder = **0**
---
### Example 3 (Negative Case)
```text
-11 = 3 × (-4) + 1
```
* Quotient = **-4**
* Remainder = **1**
---
> **Key Idea:** Every integer can be uniquely written in the form `a = bq + r`. This concept is essential for solving problems in **number theory** and **modular arithmetic**.
