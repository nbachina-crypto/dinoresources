SSL/TLS secures communication over the internet (e.g., HTTPS).
---
## Main Components
* **Handshake Protocol** → sets up secure connection
* **Record Protocol** → encrypts and sends data
---
## Basic Steps
* Client sends supported options
* Server responds with certificate
* Keys are exchanged securely
* Shared key is created
* Communication continues securely
---
## Additional Protocols
* **Change Cipher Spec** → starts encryption
* **Alert Protocol** → handles errors
---
> **Key Idea:** TLS ensures secure communication by combining **authentication, key exchange, encryption, and integrity protection** in a layered protocol design.
