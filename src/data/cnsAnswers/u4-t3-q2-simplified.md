
## Simplified Explanation
Public key distribution ensures users can **trust public keys**.
---
## Methods (Simplified)
* **Public announcement** → easy but insecure
* **Directory** → trusted storage of keys
* **Public key authority** → provides verified keys
* **PKI** → uses certificates signed by CA (most common)
---
## Key Idea
> Public key distribution is essential to prevent attackers from **replacing keys** and impersonating users, ensuring secure communication.
