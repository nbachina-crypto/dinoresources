
Firewalls control network traffic using predefined rules to **protect systems from unauthorized access**.
---
## Types (Simplified)
* **Packet filtering** → checks IP, port
* **Stateful firewall** → tracks connections
* **Proxy firewall** → inspects full data
* **Circuit-level** → checks connection setup
* **NGFW** → advanced, multi-layer protection
* **Host-based** → protects a single system
* **Network firewall** → protects entire network
---
> **Key Idea:** Firewalls enforce security policies by filtering traffic, preventing unauthorized access, and protecting systems from network-based threats.
