
A cryptographic hash function converts data into a **fixed-size hash value**.
---
## Key Uses
* **Data integrity** → detect changes
* **Digital signatures** → secure and efficient signing
* **Password storage** → protect user credentials
* **Message authentication** → verify sender and data
* **Blockchain** → secure linking of blocks
* **File verification** → ensure data correctness
* **Key generation** → derive secure keys
---
> **Key Idea:** Hash functions provide **security, integrity, and efficiency**, making them essential in modern cryptographic systems.
