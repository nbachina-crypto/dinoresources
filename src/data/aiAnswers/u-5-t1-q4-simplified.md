A **Bayesian Network** is a graph-based model used in **AI** to show how different events are related using probabilities.
It contains:
* **nodes** → variables
* **arrows** → dependency between variables
Example:
```text
Disease → Fever
Disease → Cough
```
This means **fever and cough depend on disease**.
The system stores probabilities for each relation and uses them to predict outcomes.
For example, if fever and cough are present, the **AI system can estimate the chance of disease**.
### Importance
* used for prediction
* used for medical diagnosis
* used in expert systems
* handles uncertain data
* reduces complex probability calculations
It is important because it helps **AI systems make smart decisions from incomplete information**.
