A **Constraint Satisfaction Problem (CSP)** is a problem where **values must be assigned to variables while following certain rules**.
It has **three parts**:
* **variables** – what needs values
* **domains** – possible values
* **constraints** – conditions that must be satisfied
### Example
Assign colors to regions so **neighboring regions do not have the same color**.
A **Constraint Network** is the **graph form of a CSP**.
* **nodes = variables**
* **edges = constraints**
This helps AI solve problems like:
* **map coloring**
* **scheduling**
* **Sudoku**
efficiently.
