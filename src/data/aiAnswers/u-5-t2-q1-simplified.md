### Supervised Learning
The **AI learns with correct answers**.
Example:
| Input          | Output   |
| -------------- | -------- |
| Free offer     | Spam     |
| Project update | Not Spam |
The system learns from these examples and predicts labels for new data.
Used for:
* classification
* prediction
* price estimation
---
### Unsupervised Learning
The **AI learns without answers or labels**.
It finds hidden patterns by itself.
Example:
* grouping customers with similar buying habits
Used for:
* clustering
* grouping
* pattern discovery
**Difference:**
* **Supervised = learns with answers**
* **Unsupervised = learns without answers**
