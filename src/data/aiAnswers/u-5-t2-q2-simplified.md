Decision tree learning builds a model like a **flowchart** to make decisions.
* Start with full dataset
* Choose the **best feature**
* Split data into groups
* Repeat for each group
---
## Simplified Flow
```
Start with data
Choose best feature
Split data into groups
Repeat for each group
Stop when groups are pure
```
---
## Simple Example
* First check: **Outlook**
* Then check: **Humidity**
* Finally decide: **Play / Not Play**
---
## Final Insight
**Decision Trees =**
* Repeatedly split data
* Organize into structured rules
* Make clear and interpretable decisions
They are widely used due to their **simplicity and effectiveness** in many AI applications.
