**Forward Chaining:**
* Start with **facts**
* Apply rules repeatedly
* Generate new facts
```
Start with facts → apply rules → get new facts → repeat
```
**Backward Chaining:**
* Start with **goal**
* Break into **subgoals**
* Check if facts support them
```
Start with goal → break into smaller subgoals → check facts
```
---
## Simple Insight
* **Forward Chaining = Facts → Conclusion**
* **Backward Chaining = Goal → Facts**
Both methods enable **logical reasoning in AI systems**, but are used in different scenarios depending on the problem.
