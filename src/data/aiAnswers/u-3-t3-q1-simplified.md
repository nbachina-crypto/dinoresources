A **Knowledge-based Agent** is an AI system that uses **stored knowledge and logic** to make decisions.
### Main Components
* **sensors** → collect input
* **knowledge base** → stores facts and rules
* **inference engine** → applies reasoning
* **actuators** → perform actions
### Simple Architecture
```text id="17384"
input -> knowledge -> reasoning -> action
```
It works by **sensing the environment**, using **stored knowledge**, **reasoning about it**, and then taking the **best action**.
