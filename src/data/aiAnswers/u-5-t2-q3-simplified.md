A **hypothesis** means a possible model learned from data.
To choose the best one, we test how well it predicts unseen data.
Methods used:
* **accuracy** → correct predictions
* **error rate** → wrong predictions
* **cross validation** → multiple test splits
* **precision and recall** → useful for classification
* **simplicity** → simpler model preferred
### Example
| Model   | Accuracy |
| ------- | -------- |
| Model A | 90%      |
| Model B | 85%      |
So, **Model A** is the better hypothesis.
The best hypothesis is the one that gives **high accuracy, low error, and works well on new data**.
