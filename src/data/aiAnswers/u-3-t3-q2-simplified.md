**Propositional Logic** uses simple statements that are either **true** or **false**.
### Example
* **P**: It is raining
These statements are connected using logic symbols like:
* **AND**
* **OR**
* **NOT**
**Theorem proving** means proving a conclusion from given facts using **inference rules**.
### Example
```text id="92451"
P -> Q
P
therefore Q
```
This rule is called **Modus Ponens**.
It helps AI systems **reason and derive conclusions logically**.
