The **collection phase in mobile forensics** is the process of safely gathering data from a mobile phone for investigation.
First, the device is **identified and isolated from the network** to prevent remote changes or wiping.
Then investigators create a **forensic copy** of the phone data using forensic tools.
The collected data may include:
* **messages**
* **call logs**
* **photos**
* **app data**
* **SIM data**
* **deleted files**
A **hash value** is created to confirm that the copied data is exactly the same as the original.
The **original phone is preserved**, and analysis is done only on the **copied data**.
