**File hashing** means creating a unique **hash value** for a file or disk image.
It works like a **fingerprint for digital evidence**.
If the file changes even slightly, the **hash value also changes**. This helps investigators check whether evidence has been modified.
It is mainly used to verify that the **forensic copy** is exactly the same as the original evidence.
### Filtering
**Filtering techniques** are used to sort and reduce data during analysis.
Investigators can filter files by:
* **type**
* **date**
* **size**
* **keyword**
* **deleted status**
This makes it easier to find important evidence quickly from a large number of files.
