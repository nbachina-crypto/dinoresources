**Memory analysis** means examining **RAM** to find temporary data that exists only while the system is running.
In the **collection phase**, investigators create a **memory dump** from the live system and generate a **hash value** to preserve integrity.
In the **examination phase**, they analyze the dump to find:
* **running processes**
* **open files**
* **network connections**
* **temporary user data**
This is important because RAM can contain evidence that **disappears after system shutdown**.
