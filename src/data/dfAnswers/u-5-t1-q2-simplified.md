The **Internet** is a worldwide network that connects **computers, phones, and servers** so they can communicate.
It consists of:
* **user devices**
* **routers**
* **ISPs**
* **servers**
connected through communication links.
When you open a website, the website name is converted into an **IP address**, the request is sent as small **packets**, and routers guide these packets to the correct server.
The server sends the response back, and the browser displays the webpage.
It mainly works using **TCP/IP protocols** for communication and data transfer.
