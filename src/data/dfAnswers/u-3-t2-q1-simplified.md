**Disk structure** means how data is arranged inside a storage device. It includes:
* **sectors**
* **partitions**
* **file system information**
* **file storage area**
**File system analysis** means examining how files and folders are managed in systems like **FAT** and **NTFS**.
### Basic Difference
| File System | Main Structure                         |
| ----------- | -------------------------------------- |
| **FAT**     | Uses a table to track file locations   |
| **NTFS**    | Uses the **MFT** to store file details |
During forensic investigation, this analysis helps:
* **recover deleted files**
* **check file timestamps**
* **study folder paths**
* **understand system actions**
It is useful for finding **hidden data, deleted evidence, and reconstructing the sequence of events**.
