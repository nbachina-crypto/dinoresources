**Data acquisition** means collecting digital evidence from a device safely without changing the original data. It is done to preserve evidence for investigation and legal use.
A **forensic copy** is an exact **bit-by-bit copy** of the entire storage device. It copies everything, including:
* **existing files**
* **deleted files**
* **free space**
* **system data**
This is different from a normal copy, which only copies visible files.
### Basic Steps
1. Connect the device through a **write blocker**
2. Create a **disk image**
3. Check the **hash value**
4. Verify that the copied data matches the original
The original device is kept safe, and investigators work only on the forensic copy so that the evidence remains unchanged.
